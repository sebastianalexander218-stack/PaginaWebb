# 🛒 Abarrotes Flor - Tienda Virtual

Aplicación web para la gestión de una tienda de abarrotes, con carrito de compras, panel de administración, dashboard con gráficos y registro de clientes.

## 🌐 Demo

[https://abarrotes-flor.onrender.com](https://abarrotes-flor.onrender.com)

---

## ✨ Características principales

- Catálogo de productos con ofertas y combos.
- Carrito de compras en tiempo real.
- Panel de administración con CRUD de productos.
- Dashboard con KPIs y gráficos de ventas, clientes y productos.
- Registro de clientes desde la web.
- Autenticación de administrador.
- Diseño responsive (mobile first).

---

## 🧰 Tecnologías

- **Backend**: Python 3, Flask
- **Frontend**: HTML5, CSS3, JavaScript (vanilla)
- **Gráficos**: Chart.js
- **Almacenamiento**: JSON (productos, ventas)
- **Servidor producción**: Gunicorn
- **Despliegue**: Render
