# 🛒 Abarrotes Flor - Tienda Virtual

Aplicación web de tienda de abarrotes construida con **Flask**, **JavaScript** y **HTML/CSS**. Permite gestionar productos, registrar clientes, administrar ventas y ofrecer un carrito de compras funcional. Ideal para pequeños negocios que quieren digitalizar su catálogo y ventas.

## 🚀 Demo en Render

👉 [https://abarrotes-flor.onrender.com](https://abarrotes-flor.onrender.com) *(Reemplaza con tu URL real después de desplegar)*

---

## ✨ Características principales

- 🏪 **Catálogo de productos** con ofertas y combos especiales.
- 🛒 **Carrito de compras** en tiempo real (agregar, editar cantidades, eliminar).
- 👤 **Autenticación de administrador** para acceder al panel de control.
- 📊 **Dashboard** con KPIs, gráficos de ventas y listados de clientes.
- 📦 **CRUD completo de productos** (Crear, Leer, Actualizar, Eliminar).
- 📝 **Registro de clientes** desde la web.
- 💾 **Persistencia de datos** mediante archivos JSON (productos y ventas).
- 📱 **Diseño responsive** adaptable a móviles y tablets.

---

## 🧰 Tecnologías utilizadas

| Herramienta | Propósito |
|-------------|-----------|
| **Python + Flask** | Backend y lógica de servidor |
| **Jinja2** | Motor de plantillas HTML |
| **JavaScript (vanilla)** | Interactividad en frontend (carrito, navegación, CRUD de clientes) |
| **Chart.js** | Visualización de datos en el dashboard |
| **HTML5 + CSS3** | Estructura y estilos de la interfaz |
| **JSON** | Almacenamiento ligero de datos |
| **Gunicorn** | Servidor WSGI para producción |
| **Render** | Plataforma de despliegue (PaaS) |
